# 🫀 CardioRisk AI: Heart Disease Prediction System

## 📌 Overview

CardioRisk AI is a Machine Learning-based project that predicts the likelihood of heart disease using patient health data. The goal is to assist in early detection and support healthcare decision-making.

---

## 🎯 Objective

* Build a classification model to predict heart disease
* Analyze important medical features affecting heart health
* Compare multiple ML algorithms to find the best-performing model

---

## 📊 Dataset

* Source: UCI Machine Learning Repository / Kaggle Heart Disease Dataset
* The dataset contains various medical attributes such as:

  * Age
  * Sex
  * Chest Pain Type
  * Cholesterol
  * Resting Blood Pressure
  * Maximum Heart Rate
  * Exercise Induced Angina
  * Oldpeak

---

## 🛠 Tech Stack

* **Programming Language:** Python
* **Libraries:**

  * Pandas, NumPy
  * Matplotlib, Seaborn
  * Scikit-learn

---

## 🔍 Project Workflow

1. Data Collection
2. Data Preprocessing

   * Handling missing values
   * Encoding categorical variables
   * Feature scaling
3. Exploratory Data Analysis (EDA)
4. Model Building
5. Model Evaluation

---

## 🤖 Machine Learning Models Used

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Naive Bayes

---

## 📈 Results

* Logistic Regression achieved strong performance for classification
* KNN performance improved after feature scaling
* Naive Bayes provided a good baseline model

---

## 📊 Key Insights

* Chest Pain Type is a major indicator of heart disease
* Age plays a significant role in prediction
* Feature scaling is crucial for distance-based models like KNN
