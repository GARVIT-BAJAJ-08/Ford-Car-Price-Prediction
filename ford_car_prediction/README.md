# 🚗 Ford Car Price Prediction using Multiple Linear Regression

## 📌 Project Overview

This project focuses on predicting the price of Ford cars using **Multiple Linear Regression (MLR)**, a supervised machine learning algorithm. The model learns the relationship between various car features and their prices to make accurate predictions.

The goal of this project is to understand how different factors like engine size, mileage, and year influence car prices and to build a predictive model based on these features.

---

## 🎯 Objectives

* Perform data preprocessing and cleaning
* Apply Multiple Linear Regression on real-world dataset
* Evaluate model performance using standard metrics
* Understand feature impact on car price
* Build a baseline model for future improvements

---

## 📊 Dataset Information

The dataset contains details of Ford cars with the following features:

* **Model** – Car model name
* **Year** – Manufacturing year
* **Price** – Target variable (dependent variable)
* **Transmission** – Type of transmission (Manual/Automatic)
* **Mileage** – Distance covered by the car
* **Fuel Type** – Petrol/Diesel/Hybrid
* **Tax** – Road tax
* **MPG** – Miles per gallon (fuel efficiency)
* **Engine Size** – Engine capacity

## 🔍 Project Workflow

### 1. Data Preprocessing

* Handling missing values
* Encoding categorical variables (One-Hot Encoding)
* Feature selection
* Splitting dataset into training and testing sets

### 2. Model Building

* Applied **Multiple Linear Regression**
* Trained model using training dataset

### 3. Model Evaluation

* **R² Score (Coefficient of Determination)**
* **Mean Squared Error (MSE)**
* **Residual Analysis**

---

## 📈 Results

* The model was able to capture the linear relationship between features and price
* Achieved a reasonable **R² score**, indicating decent predictive performance
* Some features like **engine size and year** showed strong influence on price
